# Panelique Building Systems — C-MAX® EPS Panels Platform

Platform for comparing traditional masonry vs C-MAX® EPS panel systems, viewing product ranges, and requesting construction quotes.

## Setup

1. `npm install`
2. Copy `.env.example` to `.env.local` and add your Gemini API key
3. `npm run dev`

## Scripts

- `npm run dev` — Start dev server on port 3000
- `npm run build` — Build for production
- `npm run server` — Start lead capture backend on port 3001
- `npm run lint` — Type check
